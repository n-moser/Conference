Conference
==========

PAC Project (JSF, EJB, JPA, JAX-RS)

Es soll eine Applikation für Konferenzen programmiert werden. Die Hauptanwender sind Gäste, die diese Applikation über das Web oder über mobile Endgeräte nutzen sollen.
